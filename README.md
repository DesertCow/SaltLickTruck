## [Installation](#table-of-contents)

----

````
npm init
 
npm install

## Configure Database Connection via Babble-Exchange/.env
## Expected Database Name = babble_DB
## CLEARDB_DATABASE_URL = "mysql://root:password@localhost/babble_DB"

npm run deploy

npm start


````
